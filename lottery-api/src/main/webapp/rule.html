利达玉米籽游戏规则说明

 

★本游戏开彩结果按照广西快乐十分当期开彩结果的最后一码（简称开出之号码）减去最接近的4的倍数值后所剩余≦4的正数值（1、2、3、4）作为本游戏的开彩结果（简称开彩结果）。如：开出之号码为15，则开彩结果为3；开出之号码为12，则开彩结果为4。

―――――――――――――――――――――――――――――

【角】即下注XY，当开彩结果为X或者Y赢，开21为和，其他的输。

【连】即X连Y，即下注X，当开彩结果为X赢，开彩结果为Y和局，

开21为和，其他的输。

如：开彩结果为1时，下注1连2、1连4赢；下注2连1、4连1为和，其他的为输。

【番】即下注X，当开彩结果为X赢，开21为和，其他的输。

【正】即下注X，当开彩结果为X赢，开彩结果为对面输，开彩结果为两边和局，开21为和。（1对3，2对4）

如：开彩结果为1时，即下注“一正”则赢，“三正”输，“二”以及“四正”为和。

【三中】即下注XYZ，当开出X、Y、Z其中一个为赢，开21为和，其

他为输。

【特码】投注号码为开出之号码，视为中奖，其余情况视为输。

【大/小】开出之号码≧11为大，≦10为小；出21为和。

【单/双】开出之号码为双数叫双，如：8、16；开出之号码为单数叫

单，如：9、15；出21为和。
